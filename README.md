# Sales-Dashboard
**Title / Headline**                                                                                                                                                    Title: Super Store Sales Analytics Dashboard.                                                                                                                            Headline: “Transforming Sales Data into Actionable Insights”.                   
**Project Description**:                                                                                                                                                This project is an interactive dashboard that analyzes superstore sales data, including sales, profit, and quantity. It uses charts to show trends by region, category, and time for better understanding.                                                                                                          
**Purpose:**                                                                                                                                                             The purpose of this project is to convert raw data into meaningful insights and help businesses make better decisions to improve sales and profit.
**💻 Tech Stack Used**                                                                                                                                                Tool: Power BI (for dashboard creation & visualization)                                                                                                                 Data Processing: Power Query (for data cleaning & transformation)                                                                                                      Language: DAX (for calculations & measures)                                                                                                                             Data Source: Excel / CSV file (Superstore dataset)                                                                                                                      Visualization: Charts, Maps, KPI Cards, Filters (Slicers) 
**📂 Data Source**                                                                                                                                                     -The data for this project is collected from Kaggle, an online platform for datasets and machine learning.                                                              -The dataset used is the Superstore Sales Dataset, available in Excel/CSV format.                                                                                       -It contains details such as sales, profit, orders, customers, region, and shipping information.  
**✨ Features and Highlights**                                                                                                                                         Interactive Dashboard: Easy-to-use interface with filters (region-wise selection).                                                                                      Key Metrics (KPIs): Displays total sales, profit, and quantity for quick insights.                                                                                      Category & Sub-Category Analysis: Shows performance of different product categories.                                                                                    Regional Analysis: Visualizes sales distribution across different regions and states.                                                                                   Time-Based Trends: Tracks sales over years, months, and days.                                                                                                           Segment Analysis: Analyzes customer segments like Consumer, Corporate, and Home Office.                                                                                 Data Visualization: Uses charts, maps, and graphs for better understanding.
**📊 Impact & Insights**                                                                                                                                             High Sales Region: West region contributes the highest sales (~32%), followed by East.                                                                               Top Category: Technology generates the highest revenue compared to Furniture and Office Supplies.                                                                    Customer Segment: Consumer segment dominates with around 50% of total sales.                                                                                         Best Sub-Category: Chairs and Phones are the top-selling products.
